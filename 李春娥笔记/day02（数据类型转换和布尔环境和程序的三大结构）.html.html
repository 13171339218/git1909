2、类型转换
自动类型转换
1、在进行算术运算和关系运算时   数字字符串会自动转成数值
2、在进行算术运算时  null会自动转成0   undefined会自动转成 NaN
7-null   7
7-undefined   NaN
强制类型转换 : 
parseInt()   强制的将某个字符串转成整数    不能正常转换结果就是NaN    【注】：遇到非数字结束转换
parseFloat()  强制将某个字符串转成小数   不能正常转换的结果就是NaN   （值也有可能是一个整数）    【注】：遇到非数字和非小数点(第二个)结束转换
Number()   强制将某个类型转成数值型   不能正常转换的结果是NaN        【注】：只能转换纯数字字符串和布尔值
Boolean()  强制的将某个类型转换布尔类型   

     console.log( Number(null) )//0    能转各种类型，但结果还是根据转换的内容来
     console.log( Number(undefined) )//NaN
     console.log( parseInt(true) )//NaN    parseInt()只能将字符串转成数字
     console.log( parseInt(null) )//NaN

toString()  将某个类型转成字符串
    如果括号中有 2或8或16 ，含义就是将十进制数转成2 或 8  或 16进制 
      
    b = b.toString();
    console.log(typeof b)//string
    b = b.toString(8);//将number类型的转成对应进制
    console.log(b)//1001

扩展 ： 
isNaN("")  判断非数字  如果括号中是非数字 结果就是true  否则就是false      数字字符串会自动转换成数值
eval("")    尝试将某个字符串转成表达式  并得到表达式的值

单分支 ：  一个条件        
多分支 ：  多个条件        
5、单分支结构
if( 表达式 ){
    语句1;
}else{
    语句2;
}
等价 ：  表达式 ？ 语句1 : 语句2
说明 ： 表达式一般为条件表达式
执行逻辑 ： 如果表达式结果为真，就执行语句1  否则就执行语句2
注意 ：
    1、else可以省略  
    2、如果if或else语句有一条，{}可以省略    多条语句{}不可以省略
在表示判断的情况下（布尔环境），下面表示真假情况 ： 
1、所有数字为真  0为假
2、所有字符串为真  空字符串为假
3、null 和 undefined 为假
4、所有对象为真
5、所有函数为真

switch()语句  --  开关语句
语法结构 ： 
switch(表达式){
    case 常量表达式 ： 语句;
    case 常量表达式 : 语句;
    ...
    default : 语句n;
}    
说明 ： 
1 、 表达式 一般是一个变量   该变量有具体的值
2、  default 可以省略  
3、  default 位置任意
4、  如果case 语句后面没有break  会实现case穿透    如果阻止case穿透，使用break阻止 
5、 switch 表达式的值和常量表达式的值进行比较时，按照全等方式进行比较的  ===   先比较类型 再比较值
执行逻辑 ： 用表达式的值和 case后面的值做比较 如果相等就执行对应的case语句
