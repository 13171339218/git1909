1.三元运算符

当你想在一行中写一个if..else语句时，这是一个很棒的代码保存器。

速记：
const x = 20;let answer;if (x > 10) {
answer = 'greater than 10';} else {
answer = 'less than 10';}
速记：
const answer = x > 10 ? 'greater than 10' : 'less than 10';
也可以像这样嵌套if语句：
const answer = x > 10 ? " greater than 10" : x

let variable2 = variable1;}

const variable2 = variable1 || 'new';

6.短路评估

我们可以简单地使用短路逻辑运算符，只用一行代码完成同样的事情，而不是写六行代码来分配缺省值。
速记：

速记：

7.十进制基数指数

你可能看过这个。它本质上是一个奇特的方式来编写没有尾随零的数字。例如，1e7实质上是1后面跟着7个零。它表示一个十进制基数（JavaScript被解释为浮点类型）等于10,000,000。
速记：
for (let i = 0; i < 10000; i++) {}
速记：
for (let i = 0; i < 1e7; i++) {}

// All the below will evaluate to true1e0 === 1;1e1 === 10;1e2 === 100;1e3 === 1000;1e4 === 10000;1e5 === 100000;

8.对象属性速记

在JavaScript中定义对象文字使生活变得更容易。ES6提供了一种更简单的方式将属性分配给对象。如果属性名称与密钥名称相同，则可以利用简写符号。
速记：
const obj = { x:x, y:y };
速记：
const obj = { x, y };

9.箭头功能速记

古典函数很容易以简单的形式读写，但一旦开始将它们嵌套到其他函数调用中，它们往往会变得有点冗长和混乱。
速记：
function sayHello(name) {
console.log('Hello', name);}

setTimeout(function() {
console.log('Loaded')}, 2000);

list.forEach(function(item) {
console.log(item);});
速记：
sayHello = name => console.log('Hello', name);

setTimeout(() => console.log('Loaded'), 2000);

list.forEach(item => console.log(item));
需要注意的是，this箭头函数内部的值的确定方式不同于longhand函数，因此这两个示例并不完全等价。

11.默认参数值

您可以使用if语句为函数参数定义默认值。在ES6中，您可以在函数声明本身中定义默认值。
速记：
function volume(l, w, h) {
if (w === undefined)
w = 3;
if (h === undefined)
h = 4;
return l * w * h;}
速记：
volume = (l, w = 3, h = 4 ) => (l * w * h);

volume(2) //output: 24

12.模板文字

你是否厌倦了用' + '多个变量连接成一个字符串？这样做没有更简单的方法吗？如果你能够使用ES6，那么你很幸运。你所需要做的就是使用反引号，并把你的变量括起来。${}
速记：

15.传播运算符速记

在ES6中引入的扩展运算符有几个用例，可以使JavaScript代码更加高效和有趣。它可以用来替换某些数组函数。扩散算子只是一系列三个点。
速记
// joining arraysconst odd = [1, 3, 5];const nums = [2 ,4 , 6].concat(odd);

// cloning arraysconst arr = [1, 2, 3, 4];const arr2 = arr.slice()
速记：
// joining arraysconst odd = [1, 3, 5 ];const nums = [2 ,4 , 6, ...odd];
console.log(nums); // [ 2, 4, 6, 1, 3, 5 ]

// cloning arraysconst arr = [1, 2, 3, 4];const arr2 = [...arr];
与函数不同，您可以使用spread运算符将数组插入另一个数组中的任何位置。concat()
const odd = [1, 3, 5 ];const nums = [2, ...odd, 4 , 6];
您还可以将展开运算符与ES6解构表示法结合使用：
const { a, b, ...z } = { a: 1, b: 2, c: 3, d: 4 };
console.log(a) // 1
console.log(b) // 2
console.log(z) // { c: 3, d: 4 }

16.强制性参数速记

默认情况下，JavaScript会将函数参数设置undefined为不传递值。其他一些语言会发出警告或错误。要强制进行参数分配，可以使用if语句来引发错误undefined，或者您可以利用“强制参数简写”。
速记：
function foo(bar) {
if(bar === undefined) {
throw new Error('Missing parameter!');
}
return bar;}
速记：
mandatory = () => {
throw new Error('Missing parameter!');}

foo = (bar = mandatory()) => {
return bar;}

18.对象[key]速记

你知道吗，也可以写成？起初，似乎没有理由为什么你应该这样写。但是，这个表示法为您提供了编写可重用代码的构建块。Foo.barFoo['bar']
考虑一个验证函数的简单例子：
function validate(values) {
if(!values.first)
return false;
if(!values.last)
return false;
return true;}

console.log(validate({first:'Bruce',last:'Wayne'})); // true
该功能完美地完成了它的工作。但是，考虑一种场景，其中有很多表单需要应用验证，但具有不同的字段和规则。建立一个可以在运行时配置的通用验证函数不是很好吗？
速记：
// object validation rulesconst schema = {
first: {
required:true
},
last: {
required:true
}}

// universal validation functionconst validate = (schema, values) => {
for(field in schema) {
if(schema[field].required) {
if(!values[field]) {
return false;
}
}
}
return true;}


console.log(validate(schema, {first:'Bruce'})); // false
console.log(validate(schema, {first:'Bruce',last:'Wayne'})); // true
现在我们有了一个验证函数，我们可以在所有表单中重用，而无需为每个表单编写自定义验证函数。
