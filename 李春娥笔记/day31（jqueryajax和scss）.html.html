1、ajax的异步请求之 load 

load() 方法是一个局部的方法，通过jquery的选择器作为开始调用load方法
load()方法是异步的，加载的信息中如果需要事件响应，必须通过回调函数来实现，也就是说所有load加载的数据中的事件都要写在回调函数中
load()方法如果想要请求不同结构的内容，可以把这些内容写到一个文件中，请求数据时，在url后加一个请求容器的选择器名称即可（这里选择器建议使用基本选择器）
        【注】：缺点---不推荐，如果文件中的数据多的话，加大了对服务器的压力，静态页面就直接写就行

load()方法的参数有三个：
第一个： 请求的url
第二个： 发送的数据  json格式  {}
第三个参数 ： 回调函数  
回调函数中有三个参数，分别是：
function(res,type,xhr){
       console.log(res);    //请求数据
       console.log( type );//请求的状态
        console.log(xhr);  //defered 对象  （类似于js中promise对象）        defered：延期的，推迟
}

     $("#headerwrap").load("loadpublic.html #header",{"第二个参数是向服务器发送的数据，是json格式，可写可不写"},function(res,type,xhr){//因为ajax是异步的（ajax.onreadystatechange状态改变事件），所以事件的添加需要在回调函数中
          //回调函数三个参数，第一个参数是接收到的数据，第二个参数是接收数据的状态，是success，还是failed
          $("#btn").on("click",function(){
              console.log(res)//因为这次请求的是div页面结构，所以收到的数据就是页面结构数据，即div#header和div#foot
              console.log(type)
              console.log(xhr)
          })
     })
     $("#footwrap").load("loadpublic.html #foot")



2、ajax请求之  $.get 或 $.post

以 $. 开始的ajax请求方式是全局方法
$.get(三个参数)   以get方式请求数据
第一个参数：请求的路径
第二个参数：请求数据　　json对象   ｛｝
第三个参数： 回调函数   （回调函数中也有三个参数，同load）

$.post()  以post方式请求数据  用法同$.get()

     $(document).on("click",function(){
          $.get("data.json",{"uname":"cdk"},function(res,type,xhr){//jqueryajax返回的数据是经过转换后的对象类型，不是string
              console.log(res.uname)
              console.log(type)
              console.log(xhr)
          });
     })


3、ajax请求之  $.getJson()      $.getScript()

$.getJson("xxx.json"，data, 回调函数) 通过这种方式请求json数据   三个参数  ： url    data     function(){}

     $(document).on("click",function(){
          $.getJSON("data.json",{"uname":"ck"},function(res){
          //三个参数，第一个参数url，第二个参数，向服务发送的数据，在控制台NetWork查看，第三个参数，回调函数
          //回调函数的参数是从服务器接收到的数据
               console.log(res)
          })
     })

$.getScript("xxx.js"，回调) 通过这种方式请求js脚本数据 ，立即执行请求的脚本代码

     $(document).on("click",function(){
          $.getScript("data.js")//通过$.getScript()请求的js脚本数据，立即执行请求的脚本代码
     });


4、$.ajax  

jquery的ajax请求服务器传递过来的数据一般都是object对象（请求的数据是object类型）

用法：
     $.ajax({
          type:"get",
          url:"http://127.0.0.1/jqAjx1706/data.json";,
          success:function(){
              console.log("成功");
          }
     });
或 $.ajax 方法返回一个defered对象  （类似promise）    defered：延期的，推迟
如果请求数据成功，通过done方法获取服务器的数据
