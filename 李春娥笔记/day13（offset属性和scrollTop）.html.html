2、scroll家族属性
scrollTop    页面垂直方向滚走的距离 【注】：低版本的谷歌识别document.body.scrollTop   documentElement:文档根节点
var scroll = document.documentElement.scrollTop || document.body.scrollTop;
scrollLeft   页面水平方向滚走的距离        【注】：scrollLeft不常用，水平滚动条不能出现，影响美观
【注】：document.documentElement.scrollTop = document.body.scrollTop = 值，可读可写 
获取页面垂直方向滚走距离 ：
    在滚走条事件中获取 ： 
        onscroll = function(){
            document.documentElement.scrollTop || document.body.scrollTop  获取页面滚走的距离
        }
    设置页面滚走的距离 ： 
        document.documentElement.scrollTop = document.body.scrollTop = 值
 
3、offset家族属性
offsetWidth / clientWidth : 前者获取  内容 + 补白 + 边框 宽度    后者 获取 ：  内容 + 补白  都可以获取非行内元素的样式值  值类型number    【注】：offset价值属性只能读，读出值类型是number类型
console.log(oDiv.offsetHeight);//72
oDiv.offsetHeight = 100;
console.log(oDiv.offsetHeight);//72
console.log(typeof oDiv.offsetHeight);//number
obj.style.width : 只能获取行内元素的样式值   纯内容的宽度   值的类型是  string
