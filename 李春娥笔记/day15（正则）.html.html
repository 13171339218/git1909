2、正则 ：  正确的规则

正则 就是 用来规范表单数据        regulation：规则，校准
正则表达式的组成 ： 普通字符 或  特殊字符 
定义 ： 
第一种定义正则表达式的方法：
    var reg = /普通字符或特殊字符/修饰符;     【注】：包含，不加其他修饰会找第一个满足条件的 
第二种定义正则表达式的方法：
    var reg = new RegExp("普通字符或特殊字符 即自己要实行匹配的规则","修饰符（i,g）");
        【注】： 传递给RegExp构造函数的两个参数都是字符串，变量类型是string的也可以
var reg = eval("/" + ch + "/g");//等价下句，等号定义的正则表达式/斜线之间不能直接写变量
var reg = new RegExp(ch,"g");//等价上句，通过RegExp构造函数定义的可以写类型为string的变量
正则中的特殊字符 :
    单字符:
    ^:   正则开始
    $:   正则结束
    . : 表示任意一个字符
    \ :  表示转义         \.表示.     
    + ： 表示其前面紧挨着的字符至少出现1次   等价 {1,}
    var reg = /^\d+$/;//只有数字
    *  : 表示其前面紧挨着的字符至少出现0次   等价 {0,}
    ？ ： 表示其前面紧挨着的字符至少出现0次   至多出现 1次       等价 {0,1}
       |    ：   表示或者  
    组合字符:
        \d  表示任意一个数字  
        \D  表示任意一个非数字字符
        \w  表示任意一个单词字符（字母或数字或下划线）
        \W  表示非单词字符
        \s  表示任意一个空白字符 
        \S  表示非空白字符        
    括号:
        {m, n}   表示括号前面紧挨着的字符至少出现m个  至多出现n个   mn为正整数
        {m}      表示括号前面紧挨着的字符只能出现m个
        {m,}     表示括号前面紧挨着的字符至少出现m个   
                [    ]     表示括号内的任意一个字符       
                    [wd3h]                    
                    [a-z] 表示任意一个小写字母       [a-zA-Z0-9] 
                [^     ]  表示非括号内的任意一个字符     [^0-9]    
                (  )  一般与或 |   连用 表示优先级
                [\u4e00-\u9fa5]   任意一个中文字符
    修饰符（标志） ： i  不区分大小写        g  表示全局查找    

3、正则中的相关方法

正则表达式相关方法
test() 检验一个字符串是否符合某一个正则规范，如果符合返回true，否则返回false
用法： reg.test(str)
----------------------------------------
exec()  根据正则表达式查找，结果会返回一个长度为1的数组 （数组只有一个值）
用法： reg.exec(str);

4、正则表达式方法练习

    str = "good good study,day day up";  
    将该字符串中的 good  替换成 best
    reg = /good/g
    str.replace(reg,"best")

    str = "  hello  big  , hello  big beg bog bag bag   ";
     将bog  beg  bag  big  替换成 bug
    reg = /b[oeai]g/g;

    去除字符串开始位置处的空白字符   
    reg = /^\s+/
        
    去除字符串结束位置处的空白字符
    reg = /\s+$/

    一次性去除字符串开始和结束位置处的空白
       reg =  /(^\s+|\s+$)/g 

    将字符串中的所有空白字符去除
    reg = /\s/g
