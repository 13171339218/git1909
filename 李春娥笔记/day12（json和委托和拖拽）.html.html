2、事件委托

委托 ：让别人去做 
事件委托 ： 某个事件让其他元素来完成  
    例如 ： 页面上有1000 个li，为每一个li添加单击事件   使用委托 ，只需要在 li的父级上 加一次事件就可以了
委托的好处 ： 
        把某个事件加到父元素上 。 提高程序的执行效率 
        动态创建的元素 可以在创建元素的函数体外部为其添加事件  

委托机制 ： 利用事件冒泡 或  事件捕获 机制 实现   
           不是所有的事件都可以实现事件委托    
获取事件源：    【注】：e.srcElement是ie的识别的获取事件源
    target = e.target || e.srcElement
委托的实现方式 ： 
    父级元素.事件 = function(){

    }
    或
    父级元素.addEventListener(事件,function(){

    })
var oUl = document.querySelector("ul");
    oUl.onclick = function(e){
    var e = e || event;//兼容获取事件对象
    var target = target || e.srcElement;//低版本（IE）使用srcElemnet，获取点击位置的元素
    console.log(target.nodeName);//target.nodeName的节点名字
    if( target.nodeName == "LI" ){//节点名字都是大写
        target.style.background = "blue";
    }
}

          作用 ： 存储数据  
 json对象定义 ： 
     var json = { 键:值,.... };       
     说明 ： 严格的json对象 键 用 双引号 引起来   【注】：严格的js环境（json文档类型中）中键必须用双引号引起来

4、拖拽 

拖拽思路 ： 
    需要的事件 ： onmousedown   onmousemove    onmouseup
    1、首先需要为拖拽的对象添加一个 onmousedown 事件
        记录 ： 鼠标点击某个对象时的内部偏移量
           var disx =  e.offsetX || e.layerX        layer：层     【注】：e.layer是火狐识别的内部偏移量
           var disy =  e.offsetY || e.layerY
         2、鼠标在文档上移动 
              要想让操作的元素动起来，该元素必须有定位
              移动的过程，实际上改变 ： 元素的left和top值
         3、 停止移动，需要出发 onmouseup事件
               鼠标抬起时，取消移动     document.onmousemove = null / ""
